### 函数性态

函数分类的基础, 为更高级的数学工具提供前提条件.

#### Monotonicity | 单调性

作用: 判断函数在区间内的增减趋势

应用面: 优化问题, 方程求解, 经济学

利用导数

#### Parity | 奇偶性

作用: 利用对称简化问题.

应用面: 积分计算, 信号处理, 物理建模

(3)连续的奇函数其原函数都是偶函数; 连续的偶函数其原函数中有且仅有一个是奇函数.

注: 设 $f(x)$连续

(1) 若$f(x)$是奇函数, 则$ $\int_{ 0 }^{ x } f ( t ) \, \mathrm { d } t$是偶函数

(2) 若$f(x)$是偶函数, 则$ $\int_{ 0 }^{ x } f ( t ) \, \mathrm { d } t$是奇函数

#### Periodicity | 周期性

作用: 描述重复规律

应用面: 波动现象, 时间序列分析,信号处理

周期函数的原函数不一定是周期函数

注:

(1)  设$f(x)$连续且以T为周期，则
$F(x) = \int_{0}^{x} f(t) \, \mathrm {d} t$是以T为周期的周期函数$\Leftrightarrow$ $\int_{0}^{T} f(x) \, \mathrm {d} x = 0$.

(2)周期函数的原函数是周期函数的充要条件是其在一个周期上的积分为零。

#### Boundedness | 有界性

作用: 有界性反映了数值的区间, 范围. 是否在 有限的范围内.

应用面: 稳定性分析, 算法收敛性, 控制理论

与 连续 紧密联系

#### Continuity | 连续性

作用: 判断函数是否无间断

应用面: 中值定理

#### 可导性

切线存在性
局部近似性

可导 意味着 具备线性近似能力



#### Differentiability | 可微性

作用: 反映函数的局部线性近似能力

应用面: 优化算法

#### 极限的性质

1 唯一性: 若极限存在, 则唯一

2 局部有界性: 去心邻域内有界

3 保号性, 保序性
